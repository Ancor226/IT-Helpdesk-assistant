 Tier 1 IT Helpdesk Assistant

A Claude-powered triage chatbot that handles common Tier 1 IT issues the way a real service desk does.

What it does:


Walks users through common issues: password resets, VPN connectivity, printer problems, Wi-Fi/network issues, asking clarifying questions before suggesting a fix, instead of dumping every possible solution at once
Tracks a live ticket status (Open → Resolved, or Escalated to Tier 2) based on how the conversation resolves
When an issue can't be resolved at Tier 1, it generates a short handoff summary, what the user reported, what was already tried, so a Tier 2 tech has full context instead of starting from scratch


Why I built it:
This mirrors the actual workflow I follow supporting 100+ users across military, medical, and enterprise environments, asking the right diagnostic questions, documenting what was tried, and knowing when to escalate rather than guessing. I wanted to see if I could translate that same judgment into a working tool.

What I'd improve with more time:
Connect it to a real internal knowledge base instead of general troubleshooting knowledge, and add an exportable transcript so the handoff summary can be pasted directly into a ticketing system.
